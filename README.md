# Skyview

Instantly generate multiple responsive image for your lazy loading. Powered by decentralized storage network Sia.

## Requirements

- Node.js
- NPM

## Install & Run

    $ git clone https://github.com/akbarnafisa/skyview
    $ cd skyview
    $ npm install

## Running the project
    $ npm run build
    $ npm run start
